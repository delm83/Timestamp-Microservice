Timestamp Microservice


Input a valid date or unix timestamp as described in the examples below

Examples:

  • [project url]/api/1983-03-09
  • [project url]/api/416016000000
  • [project url]/api/ (Current Date)

Example Output:

{"Unix":416016000000, "UTC":"Wed, 09 Mar 1983 00:00:00 GMT"}
